Paso 2

Flexbox: Tarjetas con control individual

Una tarjeta crece más que las demás para comparar comportamientos.

El nav como flex container El elemento nav.menu-flex recibirá display: flex. Esto convierte a sus hijos directos (.logo y .links) en flex items automáticamente. Agrupación de enlaces Los enlaces están dentro de un div.links secundario. Esto permite tratarlos como un solo bloque desde la perspectiva del nav, facilitando el control del espacio con justify-content: space-between. Distribución horizontal Flexbox distribuirá el espacio entre el logo y el grupo de enlaces de forma horizontal. El valor space-between llevará cada elemento a los extremos del contenedor.
